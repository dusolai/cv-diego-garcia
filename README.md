## Desplegar

- **Cloudflare Pages:** conecta este repo en el panel de Cloudflare → *Workers & Pages → Create → Pages → Connect to Git*. Sin comando de build; carpeta raíz `/`.
- **GitHub Pages:** Settings → Pages → Deploy from branch → `main` / root.

Todo el CV está en `index.html` (HTML + CSS inline). Para exportar a PDF: abrir en el navegador y `Ctrl/Cmd + P`.
